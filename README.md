# Church Management System

A comprehensive management system for churches, built to handle multiple administrative tasks in one place.

## 📋 Modules
- **Attendance (غياب)** — Track attendance for services and events
- **Absence Management (افتقاد)** — Record and follow up on absent members
- **Competitions (مسابقات)** — Organize and manage church competitions
- **Security (أمن)** — Manage security personnel and schedules
- **Services (خدمة)** — Schedule and manage church services
- **Sunday School (مدرسة)** — Manage classes, teachers, and students
- **Admin Dashboard (مدير النظام)** — Full admin control panel

## 🛠️ Tech Stack
- Python
- Streamlit
- SQLite / Database
